### For Adding New Features

1. **Copy template:**
   ```bash
   cp src/content/features/_template.md src/content/features/my-feature.md
   ```

2. **Edit frontmatter** - Fill in all fields (title, description, featureId, status, etc.)

3. **Write content** - Add markdown sections (overview, features, use cases, examples)

4. **Validate** - Run through the quality checklist

5. **Run astro sync:**
   ```bash
   bunx astro sync
   ```

6. **View your feature:** `/features/my-feature`

### For Marketing

1. **Use the template** in `src/content/features/_template.md`
2. **Follow marketing formulas** in `FEATURES-DOCUMENTATION.md`
3. **Copy tagline formula** - 5-10 words, action + benefit + proof
4. **Check checklist** - Ensure all marketing fields filled
5. **Score your copy** - Use 5-star rubric (aim for 5 stars)

### For Quality Assurance

1. **Run tests:**
   ```bash
   bun test src/tests/features/
   ```

2. **Validate content:**
   ```bash
   bunx astro check
   ```

3. **Use checklist:**
   - Reference `FEATURES-QUALITY-CHECKLIST.md`
   - 40+ items to verify
   - Gets faster with practice (10-15 min per feature)
