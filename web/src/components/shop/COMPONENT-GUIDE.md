## Animation Details

### Toast Animation (framer-motion)
```typescript
initial={{ opacity: 0, x: 100 }}
animate={{ opacity: 1, x: 0 }}
exit={{ opacity: 0, x: 100 }}
transition={{ duration: 0.3, ease: 'easeOut' }}
```

### Sticky Button Visibility
- Hidden by default
- Appears when window.scrollY > 800px
- Only visible on mobile (md:hidden)
- Smooth fade-in transition

### Countdown Timer
- Updates every second
- Shows hours:minutes:seconds
- Black/white color scheme matching theme
- Continues until time runs out

## Best Practices

1. **Images**: Use WebP format, 1200x1200px minimum
2. **Descriptions**: 2-3 sentences max for hero, detailed in features
3. **Reviews**: 3-5 featured reviews, distribute ratings realistically
4. **FAQs**: 5-7 most common questions
5. **Stock**: Set to realistic low number (5-10) for urgency
6. **Countdown**: Set to 2-3 hours for optimal urgency

## Performance Notes

- ProductHero: Static rendering (no client directive)
- FragranceNotes: Static rendering
- FeaturesList: Static rendering
- ValueProposition: Static rendering
- ProductSpecs: Static rendering
- ReviewsSection: Static rendering
- ProductGallery: client:load (critical interactivity)
- ProductHeader: client:load (dialog interaction)
- InlineUrgencyBanner: client:load (countdown timer)
- RecentPurchaseToast: client:load (animation)
- FAQMinimal: client:load (accordion)
- StickyCartButton: client:load (scroll detection)

**Result**: ~60% static HTML, ~40% interactive islands for optimal performance.
